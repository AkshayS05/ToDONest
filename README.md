## Overview

The Daily Task Tracker is a user-friendly and efficient todo app designed to help individuals keep track of their daily tasks. With a clean and intuitive interface, users can easily manage their tasks, prioritize them, mark them as completed, and make necessary updates as needed.

## Features

### Task Management

- Add new tasks to the list to keep track of upcoming activities and responsibilities.
- Edit existing tasks to update task details, ensuring accurate and up-to-date information.
- Delete tasks that are no longer relevant or completed.

### Priority List

- Add tasks to the priority list to highlight crucial activities and ensure they receive special attention.

### Task Completion

- Mark tasks as completed to keep track of accomplished goals and stay organized.

### Sorting Options

- Sort the task list based on different criteria: input order, priority, and completion status.

## Installation and Usage

To use the Daily Task Tracker app, follow these steps:

1. Clone the repository to your local machine.
2. Install the required dependencies using npm or yarn:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open the app in your web browser and start managing your tasks efficiently.

## Technologies Used

The Daily Task Tracker app is built using the following technologies:

- React: A popular JavaScript library for building user interfaces.
- TypeScript: A superset of JavaScript that adds static typing to the language.
- CSS Modules: A CSS approach that scopes styles locally to prevent global conflicts.
